![Python](https://img.shields.io/badge/Python-3.x-blue)

![Library](https://img.shields.io/badge/Library-Turtle-orange)

A classic **Snake Game built with Python using the Turtle graphics library**.
The player controls a snake that moves around the screen, eats food to grow longer, and avoids collisions with the wall or itself.

This project demonstrates **Python OOP concepts, event handling, collision detection, and file handling for high score storage**.

# 🧠 Game Logic

### Snake

* Moves forward continuously
* Changes direction with arrow keys
* Grows when food is eaten

### Food

* Appears randomly on the screen
* When snake touches food, score increases

### Scoreboard

* Displays **current score**
* Stores **highest score in `data.txt`**
